# 🎁 나소선

---

## 🔍 About

나의 소중한 사람을 위한 선물, **나소선**은 선물을 받을 사람의 정보를 기반으로 선물을 추천해주는 서비스입니다.

정보를 입력하면, 크롤링을 통해 적절한 선물 목록을 제시합니다.
